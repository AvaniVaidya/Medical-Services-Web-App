# Project Title: Health Care Service Locator Application\

## Description
A web application that helps to find nearest Health Care Service providers based on the given location. 

## Features
-Developed responsive frontend UI using React.js, featuring a search box for users to input desired city names. Implemented integration with backend services to display healthcare service locations on a map with pins based on latitude and longitude.\
-Designed and implemented Java Spring Boot backend with RESTful APIs to handle user queries. Integrated Google Places Text Search API to retrieve and filter relevant healthcare service data for frontend consumption.\
-Deployed three backend instances on AWS EC2 instances across different subnets and regions for high availability and failover resilience. Utilized Docker containers and Docker Hub for backend image management and deployment.\
-Deployed frontend instances on both EC2 and Amazon ECS with ECR for redundancy and load distribution. Configured Application Load Balancer (ALB) to efficiently manage traffic and avoid bottlenecks in the backend infrastructure.
